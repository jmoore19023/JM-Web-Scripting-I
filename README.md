# John Moore — Portfolio Site

A personal portfolio website built for Web Scripting I. The site documents my journey from the plant floor to IT and data analytics, showcasing my professional background, real-world projects, and career goals. Built with a dark theme using semantic HTML and a shared CSS design system.
